# FilmAuth Studio — UI 设计规范 v1.0

> AI影视公司内部视频生产系统

---

## 1. 设计哲学

**"内容是主角，工具是透明的。"**

影视工作者每天面对大量视频素材，UI 的职责是让内容呼吸、让操作无感。每一个像素都在回答：**这个界面是否让创作者更快地完成工作？**


### 2.2 色彩使用规则

| 场景 | 用色 | 说明 |
|------|------|------|
| 主操作按钮 | Brand-Primary (#E8753A) | 全页唯一最亮的橙色元素 |
| 视频选中态 | Brand-Primary@20% 底 + 左2px橙色边 | 不用实色，太抢眼 |
| 在线状态 | Success (#4ADE80) | 小圆点即可 |
| 紧急/过期 | Danger (#EF4444) | 标签或角标 |
| 链接/可点击文字 | Info (#60A5FA) | 下划线可选 |

### 2.3 渐变

**不使用渐变。** 但有一个例外：视频播放器下边缘可以做 1px `#E8753A@15%` 发光线，暗示"内容在此聚焦"。其余位置纯色+透明度叠加足够表达层级。

## 3. 字体

### 3.1 字体栈

```css
--font-primary: 'Inter', 'SF Pro Display', 'Noto Sans SC', -apple-system, sans-serif;
--font-mono: 'JetBrains Mono', 'SF Mono', 'Menlo', monospace;
--font-display: 'Inter', 'SF Pro Display', sans-serif;  /* 标题专用 */
```

### 3.2 字号阶梯

| 用途 | 大小 | 字重 | 行高 | 场景 |
|------|------|------|------|------|
| Display | 28px | 600 | 1.2 | 页面大标题 |
| H1 | 22px | 600 | 1.3 | 模块标题 |
| H2 | 18px | 500 | 1.4 | 区域标题 |
| Body | 14px | 400 | 1.5 | 正文、表格 |
| Caption | 12px | 400 | 1.4 | 辅助信息、时间戳 |
| Micro | 11px | 500 | 1.3 | 标签、badge |

### 3.3 中文考量

- 中文正文 14px 是最小可读尺寸，不小于此
- 标题用 600 字重（semi-bold），中文 bold 太粗会糊
- 数字和英文用等宽字体（版本号、时长、分辨率）

## 4. 间距与栅格

### 4.1 基础间距单位：4px

```
4px   ─ 微间距（icon与文字、badge内边距）
8px   ─ 紧凑间距（列表项之间）
12px  ─ 标准小间距（卡片内部padding）
16px  ─ 标准间距（模块之间）
24px  ─ 大间距（区域分隔）
32px  ─ 超大间距（页面级padding）
48px  ─ 页面顶部留白
```

### 4.2 栅格

- 侧边栏固定宽度：**240px**（收起 64px）
- 内容区：自适应，最大宽度 **1600px**（超出居中）
- 标准卡片网格：**4列**（1600px）/ **3列**（1200px）/ **2列**（800px）

## 6. 组件规范

### 6.1 按钮

| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 品牌橙实色 bg，白色文字 | 主操作（提交、新建、确认） |
| Secondary | 透明底 + 白色边框 | 次要操作（取消、返回） |
| Ghost | 透明底，文字色 | 辅助操作（筛选、排序） |
| Danger | 红色实色 bg | 危险操作（删除、拒绝） |
| Icon | 透明底 + icon | 工具栏操作 |

尺寸：大 40px / 中 32px / 小 24px
圆角：6px（统一）

### 6.2 卡片（视频卡片）

```
┌─────────────────────┐
│ ┌─────────────────┐ │
│ │                  │ │
│ │   视频缩略图      │ │  16:9 比例
│ │                  │ │
│ │        ▶ 02:35   │ │  时长标签在右下角
│ └─────────────────┘ │
│ 标题文字              │  14px, Text-Primary, 单行省略
│ 项目名称 · 3小时前    │  12px, Text-Secondary
│ 🏷️ 进行中  👤 张三    │  标签 + 负责人
└─────────────────────┘
```

- 卡片间距：16px
- Hover：Surface-4 底色 + 轻微上浮（translateY -2px）+ 阴影
- 选中态：左侧 2px 品牌橙色边 + Brand@20% 底色

### 6.3 表格

- 表头：Surface-2 底色，Text-Secondary，12px，大写字母
- 行高：48px
- 斑马纹：奇数行 Surface-1，偶数行 Surface-2
- Hover 行：Surface-3 底色
- 操作列：右对齐，Ghost 按钮组

### 6.4 弹窗/抽屉

- **弹窗**：居中，圆角 12px，遮罩 rgba(0,0,0,0.6)
- **抽屉**：从右侧滑入，宽度 480px（视频审批用 640px）
- 关闭：右上角 ✕ 按钮 + Esc 键

### 6.5 状态标签（Badge）

```
● 草稿     ── Text-Tertiary + Surface-3 底
● 待审核   ── Warning 色 + Warning@15% 底
● 修改中   ── Info 色 + Info@15% 底
● 已通过   ── Success 色 + Success@15% 底
● 已拒绝   ── Danger 色 + Danger@15% 底
● 已归档   ── Text-Tertiary + Surface-4 底
```

### 6.6 通知

- **Toast**：右上角弹出，3秒自动消失
- **Badge**：铃铛图标右上角红点/数字
- **系统通知**：侧边栏底部，重大系统消息

## 7. 动效

| 交互 | 动效 | 时长 |
|------|------|------|
| 页面切换 | 淡入 | 200ms ease-out |
| 弹窗打开 | 从中心缩放 + 淡入 | 250ms ease-out |
| 抽屉打开 | 从右滑入 | 300ms ease-out |
| 卡片 Hover | 上浮 2px + 阴影 | 150ms ease |
| 列表项展开 | 高度过渡 | 200ms ease |
| 加载态 | 骨架屏（Skeleton） | 脉冲动画 1.5s |
| 视频播放器 | 无动效 | 即时响应 |

**原则：快而不闪，动而不晕。** 所有动效 ≤ 300ms。

## 10. 暗色模式专属细节

- **视频播放器区域永远是页面最暗的区域**（#0A0A0E），比页面背景还深一级
- **文字对比度 ≥ 4.5:1**（WCAG AA），标题 ≥ 7:1
- **阴影在暗色模式下用发光替代**：`box-shadow: 0 0 20px rgba(232,117,58,0.15)`
- **图标用线条风格**（outline），不填充，减少视觉噪音
