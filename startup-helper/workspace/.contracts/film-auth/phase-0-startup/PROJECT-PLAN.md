# Film-Auth 项目计划

## 项目信息
- **名称：** Film-Auth（影视统一身份认证与权限管理系统）
- **规模：** 🟡中型
- **启动时间：** 2026-05-11
- **PRD来源：** /tmp/film-auth-prd.md

## 核心功能（V1.0）
1. 单点登录（SSO）— OAuth2授权码+JWT
2. Token机制 — access_token(2h) + refresh_token(7d) + Token黑名单（批量踢出）
3. 用户管理 — CRUD + 批量创建 + 自动生成密码 + 密码策略
4. 角色管理 — 预设super_admin + 自定义角色 + 权限模板
5. 三级权限体系 — 系统:模块:功能点 + 权限分配（勾选/JSON/模板/复制）
6. 项目级权限 — 项目成员(owner/editor/viewer) + 数据范围(own/project/all)
7. 审计日志 — 全操作记录 + 查询筛选 + 导出
8. 登录安全 — 登录历史 + 并发控制 + 验证码 + 锁定策略
9. 前端管理后台 — 9个页面（登录/仪表盘/用户/角色/权限/项目/应用/审计/个人中心）

## 技术约束
- 用户规模：初期500-1000，目标5000+
- 并发：100-500 QPS
- 登录响应 <500ms，权限校验 <50ms
- 全站HTTPS、CSRF/XSS/SQL注入防护
