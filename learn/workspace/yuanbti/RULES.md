# 鸢BTI 10维分析规则 v4

## 模型
使用 GLM-5-turbo（不用flash，质量不够）

## 角色分级与台词量
- 男主角（刘辩、孙策、傅融、袁基、左慈）→ 200句台词
- 出场多的角色 → 150句台词
- 出场少的角色 → 50句台词（不足则全取）
- 特殊角色（飞云、绣球等）→ 50句

## 10维度评分体系
S1权谋城府、S2情感表达、S3金钱务实、S4面具深度、S5行动风格、S6底线弹性、S7表达锋芒、S8温柔底色、S9权力态度、S10秩序感
- 7级：1/2/2.5/3/3.5/4/5
- S5行动风格：1=谋定后动，5=冲动先动
- 评分要有区分度

## ⚠️ 红线规则（违反=完全不可接受）

### 感情线规则
1. **所有男主的感情只指向广陵王**（玩家角色）
2. **男主之间没有任何感情线/暧昧关系**，只有敌/友、君臣、上下级
3. **男主和密探之间没有任何感情线/暧昧关系**，只有主从/上下级
4. **密探和密探之间**：部分有感情线，但必须**极其含蓄**——只暗示、不点破，用隐晦笔触

### 剧情排除规则
1. **完全排除"绒绒"相关剧情**（不引用、不分析、不提及）
2. **完全排除"魂魂"相关剧情**（不引用、不分析、不提及）

### 关系描写规则
1. 男主和他人关系只能是：敌人、朋友、君臣、上下级、同盟、利用、对抗
2. **禁止**使用"羁绊""守护""依赖""依恋"等词形容男主之间的关系
3. 禁止把任何角色关系写成超越普通人际关系的情感

### 绝对化词汇规则
1. 对广陵王可以用"最""唯一"等绝对化表述（感情确实是这样的）
2. 对其他任何人/事的描述，**禁止用"唯一""最"等绝对化词汇**
3. 改用"特别""很""格外""尤其""相当"等程度副词
4. 示例：❌"刘协是他唯一不愿伤害的人" → ✅"刘协是他不太愿意伤害的人"

## 写作要求
- ❌ 去AI味：禁止"值得注意的是""不难发现""综上所述""展现了""体现了"等套话
- ✅ 用大白话，像玩家写repo
- 剧情精简，用人话概括
- 每维度3-5条台词引用（来自数据库查询）
- 情感共鸣：写角色最打动人的地方
- 台词要覆盖全部时期，不要只引用早期台词

## 特殊角色规则
- 飞云、绣球、黄盖等用幽默风格
- 台词引用用"翻译"方式
