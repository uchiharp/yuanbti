## 评分体系
- **10维度**：S1权谋城府、S2情感表达、S3金钱务实、S4面具深度、S5行动风格、S6底线弹性、S7表达锋芒、S8温柔底色、S9权力态度、S10秩序感
- **7级**：1(极低) / 2(低) / 2.5(偏低) / 3(中) / 3.5(偏高) / 4(高) / 5(极高)
- **S5行动风格**：1=谋定后动，5=冲动先动

## 每个角色的评分流程
1. `db9 sql learn_test -q "SELECT text FROM dialogues WHERE speaker = '角色名' AND length(text) BETWEEN 10 AND 150 AND text NOT SIMILAR TO '%（%）%' AND text NOT LIKE '%【%' AND text NOT LIKE '%头像%' AND text NOT LIKE '%心纸君%' AND text NOT LIKE '%离魂%' AND text NOT LIKE '%ref%' ORDER BY random() LIMIT 50;"`
2. 基于50条台词样本，按10维7级评分
3. 每个维度写评分依据（附1-3条原文台词引用）
4. 写入产出文件
