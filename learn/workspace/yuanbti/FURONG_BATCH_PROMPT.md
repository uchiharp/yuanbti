# 傅融批次出题 Prompt（v1 — 基于台词深度分析版）

## ⚠️ 核心易错点（出题前必读）

### 傅融最大的陷阱：把"话少"理解成"冷漠无情"
- 傅融**话少但行动力极强**。他不说的部分，全用做了来表达
- "上位者，疑心病最好别太重" — 这是在关心广陵王，但用命令式语气
- "别动，还想不想进宫点卯了？" — 帮她整理仪容时说的话，嘴上凶实际体贴
- 他给流浪狗取名"飞云"、偷偷给它付钱、嘴上说"等它伤好了就送走"但一直养着 → 典型的嘴硬心软

### 傅融的第二大陷阱：把他写成"没有情感的工具人"
- 他有明确的情感，只是不擅长表达
- "今夜想问你，你打算这样，到什么时候？" — 他对广陵王女扮男装的身份有担忧，这是在关心她的未来
- "保证过……会陪你……走下去的……" — 受伤时说的话，字字重情
- "有时候忙完了，会来这一待会儿" — 他有自己独处思考的习惯，不是完全的机器

### 傅融的第三大陷阱：权谋题搞错他的立场
- 傅融是绣衣楼校尉，忠于朝廷/广陵王，**不是**忠于某个士族
- 他和袁基不同——袁基有家族包袱，傅融的立场更简单：保护广陵王
- 但他也有自己的判断——"已经中计，现在撤退也来不及了"说明他有独立思考
- 他不是无脑执行命令的人，但也不会违抗广陵王的决策

## 傅融人设（严格遵守）

### 核心定位：外冷内热/忠诚守护型
- **表面**：寡言少语、冷淡、公事公办
- **实际**：极度忠诚，保护欲是本能，用行动代替语言
- **嘴硬心软**：嘴上嫌弃狗（"它是狗，我是傅融"），实际上给它付钱、取名、遛弯
- **不擅表达**：想关心你的时候会突然说"跟我去个地方，有些话想同你说"
- **务实**："少来。记得报销这次的车马费" — 即使在严肃场合也不忘报销
- **受伤时的脆弱**：丹书白马/25是他重伤的一章，"还不如上一次……至少……没被你看见这副样子"——他不想被广陵王看到自己虚弱的样子

### 傅融的情感逻辑
- 他的关心藏在细节里——提醒发工资、帮你编病假、遛狗时问你要不要一起去
- 他对广陵王的感情：从上下级的忠诚，逐渐变成了超越上下级的在意
- 他不会直接说"我喜欢你"，但会问"你打算这样到什么时候"——这是他在试探广陵王的未来规划里有没有他
- 面对危险，他永远是第一个挡在前面的人

### 关键名场面
1. **乌云白云·狗（飞云）** — 捡到流浪狗，嘴上嫌弃实际照顾，经典嘴硬心软
2. **丹书白马/07** — 曹操诱敌，傅融冷静分析局势
3. **丹书白马/25** — 重伤，不想被广陵王看到虚弱，但保证过会陪她走下去
4. **城楼夜谈** — 高处独处时问广陵王的身份和未来
5. **进宫点卯** — 帮她整理仪容，嘴上凶实际体贴

## 出题规则

### 基础规则
- 维度key只用S1-S10数组格式，每题覆盖2-3个维度
- 分值只用1/3/5
- 选项完整主谓宾，禁止括号补充动作
- A保守/B忠于原文/C更大胆
- 台词从数据库拉，不编造

### 选项铁律
1. **选项是广陵王（或代入角色）的行为选择，绝对不能塞傅融的回应**
2. **选项是具体行动，不是观点/评价/分析**
3. **傅融的话少，但选项不能因此也写成沉默选项** — 要写具体行动

### 题干铁律
1. **简单场景3句话内，复杂场景允许4-5句话**
2. **点明角色名字，不写"他"**
3. **不提章节名**
4. **广陵王视角为主，"假如你是傅融"为辅**
