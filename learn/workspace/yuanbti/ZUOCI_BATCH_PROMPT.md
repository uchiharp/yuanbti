# 左慈批次出题 Prompt（v1 — 基于台词深度分析版）

## ⚠️ 核心易错点（出题前必读）

### 左慈最大的陷阱：把他写成"佛系老头/高冷仙人"
- 左慈**看起来什么都不在乎**，但**实际上在乎一切**——他只是看得太透了
- "除了吾，不要相信任何人"——这不是佛系，这是极致的保护
- "雪球一旦开始从山顶滚落……无论你抱有多大的意志和勇气张开双臂，都无法阻挡它的滚落"——他看清了残酷的真相，但选择站在广陵王这边
- "吾会站在你这边。会替你实现你想做的事的"——他不是旁观者，他是选择下场的仙人

### 左慈的第二大陷阱：把他写成"全知全能的神"
- 左慈**有弱点**——"吾……不擅饮酒"喝半口就倒，说明他并非刀枪不入
- "有时，吾会忘记自己的名字"——活了太久，记忆在流失，这是他真实的不安
- "羽衣和身披羽衣的人，都已经老了"——他对时间流逝有清醒的认知
- "有你在的人间，左慈便在"——他需要广陵王作为锚点，不然他可能真的忘记自己是谁

### 左慈的第三大陷阱：权谋题立场搞错
- 左慈**不参与世俗权力斗争**，但**对权力有极深的理解**
- "权力是石头，你把它裹上丝绸也好、染成彩色也好，它的本质都是石头。拿起石头，先让他们恐惧你，才能让他们爱你"——这是他教广陵王的权谋课
- 他对里八华的分析极其精准（千年组织、伪装渗透离间），说明他的情报能力和分析能力远超常人
- 他的"不参与"是选择，不是无能为力

### 左慈的第四大陷阱：师生关系写成"恋爱"
- 左慈和广陵王的关系本质是**师徒**
- "吾是你师尊！""以后吾不在，不要喝来路不明的酒"——这是师长的关心
- 但他的关心**超越师徒**——"真想看你穿上它的那日""和你一起看的这片海，吾会一直记得"
- 这种"超越"不是恋爱，是一种更深的、跨越时间的陪伴——他可能等了广陵王好几世

## 左慈人设（严格遵守）

### 核心定位：超然守护/深不可测型
- **表面**：慵懒、随性、什么都不在乎、说话慢半拍
- **实际**：洞察一切、掌控全局、选择站在广陵王身边、对她的保护超越了师徒
- **时间感**：活了太久，经历过沧海桑田，对时间有自己的感知——"吾记得，在很久之前，这片沧海曾是桑田"
- **表达方式**：用"吾"自称、说话简洁但字字有分量、偶尔用感叹号（"好"）

### 左慈的关键矛盾
1. **超然 vs 在意** — 他应该超然物外，但他选择在意广陵王
2. **全知 vs 无力** — 他知道一切，但"雪球一旦滚落，无法阻挡"
3. **长生 vs 遗忘** — 活了太久反而开始遗忘，需要广陵王作为锚点
4. **师尊 vs 更深** — 师生关系是外壳，里面是一种说不清的跨越时间的陪伴

### 左慈的独特细节
- 辟谷不食（"吾辟谷，不食炙肉，你吃吧"），但会给广陵王烤肉
- 喝半口酒就倒（"吾……不擅饮酒"）
- 辨别古董的能力极强（一眼看出染色的玉石是假的、松石链子是假的）
- 对自然有特殊的感受——不喜欢修剪的盆景（"过于矫揉造作"），认为"万物的天然之美就很好"
- 钓鱼时极有耐心（"再等一等，总会上钩的"——和孙策形成鲜明对比）

### 关键名场面
1. **东郡太守/01-06** — 揭示里八华的存在，教广陵王"除了吾，不要相信任何人"
2. **古艳歌** — 被当街指控时的反应、脱法袍制药、展示天人羽衣、"有你在的人间，左慈便在"
3. **乌飞** — 带广陵王去找复文箓、"乱世恩仇，皆是明月芦花"、"十四岁时，你离开吾"
4. **七夕欢情** — 投壶连中十支被老板赶走、钓鱼等一整夜、下棋赢六个月收益
5. **犬都纪事** — 对权力的深刻分析、"抱着成为下一个公敌的觉悟，做你觉得正确的事吧"

### 台词风格参考
- "好。"（左慈最常用的词，简短有力）
- "走罢。去哪里都可以，吾都陪你。"
- "乱世恩仇，皆是明月芦花。唯有吾，无论发生何事，都会站在你那一边。"
- "梅树能活很久，比绝大多数人都久。只要活下去，总有一天能恢复自由。"
- "你问这东西对身体好不好？易致脾胃虚寒，但喝了心情会好。饮食只要能让人身心愉悦就好，不必纠结太多。"

## 出题规则

### 基础规则
- 维度key只用S1-S10数组格式，每题覆盖2-3个维度
- 分值只用1/3/5
- 选项完整主谓宾，禁止括号补充动作
- A保守/B忠于原文/C更大胆
- 台词从数据库拉，不编造

### 选项铁律
1. **选项是广陵王的行为选择，绝对不能塞左慈的回应**
2. **选项是具体行动，不是观点/评价/分析**
3. **左慈的场景选项可以偏安静/内敛，但不能写成"什么都不做"**

### 题干铁律
1. **简单场景3句话内，复杂场景允许4-5句话**
2. **点明角色名字，不写"他"**
3. **不提章节名**
4. **广陵王视角为主，"假如你是左慈"为辅**

### 权谋题特别提醒
- 左慈的权谋理解是"上帝视角"级别——他看透了权力的本质
- 出权谋题时，要体现他的"教"——他在教广陵王如何运用权力
- 不要把他写成参与者，他是导师/观察者/最后的后盾

## 称呼
左慈、先生、师尊
