# 孙策批次出题 Prompt（v1 — 基于台词深度分析版）

## ⚠️ 核心易错点（出题前必读）

### 孙策最大的陷阱：把他写成"莽夫/无脑"
- 孙策**冲动但不愚蠢**。"已经中计，现在撤退也来不及了"——等等，这是傅融的台词
- 孙策的判断力体现在战场直觉上："怕的话就叫出声，我不笑你""很快就会结束的"
- 他对权力的认知非常清醒："权力是石头""权力对我来说太可怕了"——他追求的是天下太平，不是权力本身
- 他能看出士族的虚伪："公敌是可以被塑造的。士族想要延续这个公敌，避免犬都开始内斗"（绒绒版，但人设一致）

### 孙策的第二大陷阱：把他写成"只会打打杀杀"
- 他有温柔的一面："我替你切肉。你今天的妆好看，肯定用心画了，不想弄花"
- 他有幼稚的一面：连输十七把五子棋、吃饵糕烫舌头、把捕蝇草惹炸了
- 他有深沉的一面："你若想走，我护你走"——围城时的承诺，生死关头的真诚
- 他对大乔（绒绒版对广陵王）的感情是热烈的，但不是霸道的——"你想走吗？你若想走，我护你走"

### 孙策的第三大陷阱：权谋题立场搞错
- 孙策**不屑权谋**但**不是不懂权谋**——他知道士族在搞什么，他选择不跟他们玩
- "绝大多数的犬心里想的还是，自己想要天下和权力吧"——他对人心有清醒的认知
- 他的政治立场：打天下是为了太平，不是为了当皇帝。所以他帮广陵王守城是真的在帮她，不是在投资
- 围城剧情中，他一人力战守城，不是为了邀功，是因为"我不怕。我只怕死得憋屈"

### 孙策的第四大陷阱：搞笑题写成"小学生"
- 孙策的搞笑是**性格带来的反差萌**，不是低幼
- 江东小霸王套圈套不到、钓鱼钓不到、五子棋连输十七把——但这些都是七夕活动剧情里和广陵王的约会场景
- 他把九连环直接劈开（"大力出奇迹"）、把鲁班锁劈二——这才是孙策式搞笑：暴力解决问题

## 孙策人设（严格遵守）

### 核心定位：热烈直球/豪爽守护型
- **表面**：大大咧咧、冲动、豪爽、爱打架
- **实际**：心思细腻（会注意到对方妆容）、极度护短（整夜守在客房外）、对爱情直球（"大丈夫为心上人死，死得坦坦荡荡"）
- **反差萌**：战场猛将 + 日常生活笨蛋（不会钓鱼、不会下棋、被五子棋虐）
- **说话风格**：直来直去，常用"老子"、语气助词多（"啊""嗝""唔"），偶尔江东话（"赤佬"）

### 孙策的情感逻辑
- 对喜欢的人：热烈追求、直球表达、"老子今夜可是能为心上人去死的"
- 面对危险：第一个冲上去、"有我在，你什么都不用怕"
- 受伤后：不在乎自己的伤、反而担心对方被吓到、"别弄脏了……等明天，你还要穿着这身常服，毫发无伤地站出去呢"
- 对喜欢的人的未来规划："把那套什么广陵王妃的礼服，照我的尺寸改改"——他的求婚方式

### 关键名场面
1. **围城（恋念剧情）** — 一人守城，"大丈夫为心上人死，死得坦坦荡荡"；受伤后"我命硬""左臂有点……抬不起来了"
2. **南有乔木/15** — 整夜守在客房外，"我亲自守着，安心吧"
3. **南有乔木/22** — 游猎遇山贼，"怕的话就叫出声，我不笑你""有我在"
4. **七夕欢情** — 投壶连中二十支、钓鱼用金块换徒手抓、五子棋连输十七把、吃饵糕烫舌头
5. **犬都纪事** — "权力是石头，拿起石头，先让他们恐惧你，才能让他们爱你"（这是左慈说的，孙策说的是"权力对我来说太可怕了"——注意区分！）

### 关键纠正
- 孙策说"权力太可怕了"，不是说不要权力，是说他打天下不是为了权力
- 孙策的"老子"是口头禅，不是粗鲁，是他的豪爽性格
- 孙策面对喜欢的人不是霸道，是热烈+细腻

## 出题规则

### 基础规则
- 维度key只用S1-S10数组格式，每题覆盖2-3个维度
- 分值只用1/3/5
- 选项完整主谓宾，禁止括号补充动作
- A保守/B忠于原文/C更大胆
- 台词从数据库拉，不编造

### 选项铁律
1. **选项是广陵王的行为选择，绝对不能塞孙策的回应**
2. **选项是具体行动，不是观点/评价/分析**
3. **搞笑题的选项可以轻松，但不能低幼——要像真人会说的话**

### 题干铁律
1. **简单场景3句话内，复杂场景允许4-5句话**
2. **点明角色名字，不写"他"**
3. **不提章节名**
4. **广陵王视角为主，"假如你是孙策"为辅**
