# 鸢BTI 出题指南（子Agent专用）v5

## 你的任务
为指定角色出题，每道题必须基于真实剧情/台词，必须先阅读完整个人物剧情和上下文再出题。

## 一、题目结构

- 每题3个选项（A/B/C），分值 **1/3/5**
- 每题只测 **1个维度**
- 每个选项必须有 **reasoning**
- 6种type：sweet/funny/angst/scheme/daily/classic
- JSON格式，dimension用**中文key**（如`["情感表达"]`），scores同理（如`{"情感表达":5}`）

## 二、10维精确定义（⚠️ 最关键，出题前必读）

| 维度 | 高分(5)= | 低分(1)= | ⚠️ 常见误判 |
|------|---------|---------|------------|
| S1权谋城府 | 深谋远虑、算计布局、隐藏真实意图 | 坦率直来直去、不藏心思 | ❌果断≠权谋 ❌冷静≠权谋 |
| S2情感表达 | 直球表达感情、不掩饰情绪 | 回避情感、假装无感 | ❌大胆≠情感表达 ❌主动≠表达 |
| S3金钱务实 | 精打细算、看重物质回报 | 淡泊金钱、不在意利益 | ❌节俭≠务实 |
| S4面具深度 | 刻意伪装、人前人后两副面孔 | 率真、不隐藏真实想法 | ⚠️只用于"伪装/隐藏真实面目"的场景 ❌看穿别人的伪装≠S4（那是S1洞察力） ❌配合对方演戏≠S4（要看答题人自己是否也在伪装） |
| S5行动风格 | 冲动先动、想到就做 | 谋定后动、谨慎等待 | ❌果断≠行动力 ❌行动升级≠锋芒 |
| S6底线弹性 | 底线灵活、必要时不择手段 | 有绝对红线、绝不越界 | ❌迷茫≠灵活 |
| S7表达锋芒 | 嘴毒、尖锐、当众怼人/拆穿、不给面子 | 温和圆滑、给面子、回避冲突 | ❌大胆≠锋芒 ❌勇敢≠锋芒 ❌关心≠锋芒 ❌果断≠锋芒 |
| S8温柔底色 | 温柔体贴、体谅他人感受 | 冷漠疏离、不在意别人感受 | |
| S9权力态度 | 积极介入权力博弈、追求权力 | 放弃/回避权力、不关心权力 | ❌霸道≠权力态度 ❌命令≠权力 |
| S10秩序感 | 讲规矩、有条理、按规则来 | 随性散漫、不拘小节 | |

### 维度匹配硬性规则
1. **出题前必须确认**：这个场景的三个选项，在目标维度上是否有明显的梯度差异？
2. **不能把其他维度的高分行为误贴为目标维度**：
   - 大胆/勇敢 ≠ S7表达锋芒（没有攻击性就不是锋芒）
   - 果断/主动 ≠ S5行动风格（没有"冲动vs谨慎"的对立就不是行动风格）
   - 关心/温暖 ≠ S2情感表达（没有"表达vs回避"的对立就不是情感表达）
   - 坚定/强势 ≠ S9权力态度（和权力博弈无关就不是）
   - 大胆回应好感 ≠ S7表达锋芒（是S2情感表达）
   - 看穿别人伪装/拆穿别人面具 ≠ S4面具深度（是S1洞察力/权谋）
   - S4面具深度只测答题人自己是否伪装真实想法，不测能否看穿别人
3. **如果一个场景在多个维度上都有梯度，选最自然的一个**
4. **如果一个场景在任何维度上都没有明显梯度，不要出这道题**

## 三、选项设计

1. **从场景自然生长** — 三个选项是面对同一情境的3种合理反应
2. **A=1 B=3 C=5的固定排列不超过总题量5%** — 其他题目分值排列要有变化
3. **选项是具体行动** — 不是观点/分析，不以"你"开头
4. **选项之间有明显差异** — 不能只是措辞不同
5. **场景基调匹配** — 搞笑场景选项也要搞笑，angst不能太轻松
6. **如果选项是答题人做出选择/说话，题干中要有明确提示**（"你会说…""你会选择…"）
7. **任何type都可以有搞笑/出人意料的选项**，不限于funny类型

## 四、reasoning强制格式

三步推导，不允许跳步：
```
① 选项的具体行为（引用选项原文关键词） → ② 该行为在目标维度的含义 → ③ 分值
```

- ✅ 正确：`「凑过去帮他整理花枝」，主动回应好感 → 情感表达:5`
- ✅ 正确：`「当众拆穿他的借口」，公开怼人不给面子 → 表达锋芒:5`
- ❌ 错误：`大胆回应 → 表达锋芒:5`（"大胆"不是锋芒的定义）
- ❌ 错误：`务实分析 → 金钱务实:5`（没有说清楚为什么务实）
- ❌ 错误：reasoning描述的行为和选项文本对不上

## 五、题干要求

1. **3句话内**，点明角色名，不写"他"
2. **不提章节名/活动名/剧情线标签**
3. **必须有足够场景信息** — 读者能知道是谁、什么情境
4. **题干不含其他角色台词** — 题干是背景描述不是对话记录
5. **不使用绒绒/动物剧情** — 含"喵/汪/围脖/嗷呜"的必跳过
6. **台词说话对象必须是广陵王，广陵王必须在场**

## 六、内容要求

1. **先阅读完整个人物剧情+上下文，再出题** — 不能看一句台词就出
2. **台词从数据库拉**，必须理解真实含义（查上下文前后10行）
3. **不OOC** — 广陵王不对男主说冷漠粗鲁的话
4. **禁止AI味** — 不用"值得注意的是""综上所述"等套话
5. **有梗有趣** — 去掉角色壳子就不成立的不要出
6. **梗密度要求** — ⚠️ **硬要求≥40%**，每批题中至少40%要融入梗元素，包括但不限于：
   - **代号鸢梗**：角色经典台词/名场面/口头禅（如傅融"图钱呗"、袁基"在下不擅投壶"）
   - **新三国梗**："我从未见过有如此厚颜无耻之人"等经典句式改编
   - **互联网梗**：打工人、房贷、社畜、PUA、00后整顿职场等现代语境
   - **含蓄性暗示/暧昧**：不粗俗但能让人会心一笑的双关（如傅融"流氓登徒子"式害羞）
   - 梗可以出现在题干、选项或reveal中，但不影响维度测试的准确性
   - **反面教材（硬凑梗）**：
     - ❌ "你看那朵烟花像不像傅校尉加班时的黑眼圈"——刻意联想，不像人说的话
     - ❌ "帮你把炙肉留一份，做记号防张辽偷吃"——为搞笑而搞笑，和情境无关
     - ❌ "这件袖口比上次短了半寸，针线活退步了啊"——回避情感的方式太刻意
   - **好梗的标准**：信手拈来、情境自然、让人会心一笑，不是刻意搞笑。比如傅融砍价"两千零五十，停，我们转身"这种角色本身自带的就是好梗

### 选项语言要求
- **每个选项都要有画面感和灵气**，读起来像角色会说的话，不像测试题的标准答案
- **反面教材（干瘪选项）**：
  - ❌ "你也去歇着吧，天塌不下来"——正确但没味道
  - ❌ "说帐册的事不急，先歇歇"——功能性的，没有画面感
  - ❌ "说好的，西凉势力本来也是隐患"——太标准答案了
- **好选项的标准**：像"金子又不认识自己住哪箱"——有画面、有性格、读完会笑。不一定用原文，但要原文那种舒服的语感
- **每句话都要有情商**，不落俗套。广陵王是高情商的人，她的回应应该让人觉得聪明，不是让人觉得正确
6. **安慰场景**广陵王不能说太冷的话（但贾诩等角色特色冷漠保留）
7. **情感包袱检查** — 题干不应加入与选项梯度无关的情感渲染（如负罪感、道德压力）。例如：测S3金钱务实时，题干加了"平民百姓没人在乎"，会让"让他休息"变成冷漠选择，破坏梯度。题干只提供维度测试所需的信息
8. **题干-选项一致性** — 选项必须是对题干情境的自然回应，不能题干说A、选项回应B。例如：题干是"下雨天不好的事多"（脆弱倾诉），选项却在讨论"该不该伤害无辜"（道德底线），完全脱节
9. **选项逻辑自洽** — 选项内的因果链必须成立。例如："放了，她的情报比她的命值钱"——放了和拿到情报之间没有因果关系，逻辑断裂。每个选项的理由必须能推导出行动
10. **广陵王人设底线** — 广陵王情商很高，选项不能太没礼貌/不识好歹/OOC。低分选项（S8冷漠、S1坦率等）是冷淡或直率，不是蠢或没教养。例如：别人帮你解头发还说"不用你解"就是OOC，改成"嗯手艺不错"然后自己接手才是冷漠但不失礼
11. **广陵王是上级** — 傅融是绣衣楼副官，广陵王是主人。选项不能出现广陵王帮下属做他分内的事（写自辩文书、替他汇报、替他对质等）。广陵王能做的是：暗中安排人手、下命令调动资源、替他出头摆平事，不是亲力亲为替他干活
12. **敏感话题** — 生育、怀孕等话题容易引发争议，出题时避免。同理避免涉及政治敏感、宗教敏感内容

## 七、分值方向一致性

同一维度方向必须全局一致：
- S6：底线灵活/不择手段=5，有绝对红线/绝不越界=1
- S9：积极追求权力=5，回避放弃=1
- S1：深谋=5，坦率=1
- S8：温柔=5，冷漠=1
- **不能出现方向反转**
- **⚠️ 常见错误：S6容易把"道德高尚"误设为高分，实际S6高分=底线灵活。审核时逐题检查S6的5分选项是否真的是"灵活/突破底线"而非"坚守原则"**

## 十二、选项格式要求（⚠️ 常被违反，必须遵守）

1. **不要引导语** — 选项直接放台词或动作描述，不要「看着他认真说」「笑着说」「直接说」「对他说」「叹口气说」这类引导语。干巴巴放一句话上去，更有趣。
2. **不要多余心理描写总结** — 破折号后面不要加总结性描述，如「——精打细算地把他的钱也算了」「——当众怼人不留情面」「——这就是温柔」。如果破折号后面是对话内容或具体动作（有画面感的），可以保留。
3. **锋芒不只是正面怼** — 表达锋芒（S7）的选项可以多样化：
   - 侧面绕过：你说你的，我聊别的，但让你下不来台
   - 顾左右而言他：假装聊别的，实际上在嘲讽
   - 找对面别的弱点：不回你的攻击，戳你另一个痛处
   - 沉默+一个动作：不说话，做了什么比说什么更扎
   - 反问：不回答，反问你一句让你自己打脸
   - ⚠️ 不要三个选项都是正面回应同一个点
