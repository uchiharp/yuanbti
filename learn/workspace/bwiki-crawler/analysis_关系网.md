# 代号鸢角色关系网分析

## 分析方法

基于 **2593 个剧情页面**（密探故事 1098、活动剧情 366、红鸾花笺 295、密探传唤 239、恋念之音 223、主线剧情 150、约会 63、恋念剧情 29 等）和 **116,470 条对话**数据，通过角色共现频率和对话内容分析构建关系网。

共现定义为：两个角色在同一剧情页面（story_page）的 characters 数组中同时出现。排除通用角色（我、众人、广陵王、侍从、侍卫、群众角色等）后，对 50+ 核心角色两两计算共现次数。关系类型通过对话原文中的称呼、语气、互动模式判断。

广陵王的五个男主线中，角色对她的称呼呈现明显差异：

1. **袁基线**：以"殿下"为主（655次），极尽温柔与礼数，"殿下"二字既是尊称也是爱称
2. **刘辩线**：直呼"广陵王"（297次），带有强烈的占有欲和亲密感，强调"我的"
3. **孙策线**：几乎不称殿下，直来直去，体现孙策不拘小节的性格
4. **左慈线**：称"广陵王"或"吾的弟子"，师尊身份明确，语气超然
5. **傅融线**：几乎不用敬称，用"你"直接对话，最平等的关系

### 辟雍学宫圈

**成员**：郭嘉、贾诩、荀彧、荀攸、戏学、陈登、程昱、徐庶、史子眇、孔融、张邈、袁基、董奉、诸葛瑾、诸葛诞、杨修、凌统（晚辈）

**核心关系**：
- **同窗三人组**：郭嘉、贾诩、程昱——学宫时代最亲密，互称"学长"
- **徐庶-史子眇-陈登**：同窗兼绣衣楼核心
- **荀彧-郭嘉-贾诩**：学宫中的复杂三角，荀彧为"荀学长"
- **诸葛瑾-诸葛诞**：兄弟同在学宫

**关系特征**：学宫同窗关系是代号鸢世界最核心的社会网络，几乎所有重要谋士都有学宫背景。学宫关系超越了阵营对立，即使各为其主，同窗情谊仍在。
