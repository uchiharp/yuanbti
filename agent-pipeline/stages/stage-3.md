# 阶段3：UX 设计

## 任务
基于 PRD 和架构方案，产出用户体验设计。

## 必读文件
1. PRD.md
2. ARCHITECTURE.md

## 产出物
| 文件 | 说明 |
|------|------|
| UX-DESIGN.md | 用户体验设计方案 |
| ux-review.md | PM 审查意见 |
| re-review-ux.md | PM 复审确认 |

## 执行流程
1. 读 PRD.md + ARCHITECTURE.md
2. 产出 UX-DESIGN.md
3. PM 对照 PRD 审查：页面覆盖度、交互完整性、操作路径、逻辑 bug
4. UX 根据意见修改
5. **PM 复审**：确认问题已修复
6. 复审通过 → 推进阶段 3.5；仍有问题 → UX 再改（最多 2 轮）

## 约束
- 总轮次 ≤2
