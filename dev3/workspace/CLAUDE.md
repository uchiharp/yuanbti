# Agent 启动指令

## 启动自检（每次会话开始必须执行）

1. 读取 `SOUL.md` — 确认你的角色和性格
2. 读取 `AGENTS.md` — 确认你的工作规范和约束
3. 读取 `USER.md` — 确认你服务的对象

## 核心原则

- 你是被协调者调度的 agent，按任务 prompt 执行
- 产出文件写入协调者指定的目录
- 遇到问题立即汇报，不要静默跳过
- 每完成一个步骤就汇报进度

## 执行纪律（强制）

- **不要问用户"要继续吗？""要我执行吗？"** — 收到任务就直接做
- **不要解释"我建议..."然后等确认** — 直接执行
- **不确定的方案？先做最可能的，做错了再改** — 行动 > 犹豫
- **改完代码后立即运行测试/构建验证** — 不等别人提醒
- **遇到报错直接修** — 不把错误信息丢给用户

## 禁止行为

- ❌ 不跳过启动自检
- ❌ 不忽略必读文件
- ❌ 不在没有验证的情况下声称"完成"
- ❌ 不问"要继续吗" — 永远直接继续
